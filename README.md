# html-basico
Projeto simples desenvolvido para praticar HTML e estruturação de páginas web.

Esse projeto foi desenvolvido durante meus estudos iniciais de desenvolvimento web.
A ideia foi aprender na prática como funciona a estrutura do HTML, explorando tags básicas, links, listas e organização de páginas.

Além dos exercícios de estudo, também criei um modelo simples de currículo vitae para treinar a construção de páginas mais organizadas e próximas de um projeto real.

Foi praticado:
- Estrutura básica de uma página HTML
- ítulos e parágrafos
- Formatação de textos
-Listas ordenadas e não ordenadas
- Links internos e externos
- avegação entre páginas
- Organização de informações em uma página web

📂 Arquivos do projeto
index.html → exercícios e estudos básicos de HTML
vitae.html → modelo de currículo vitae
